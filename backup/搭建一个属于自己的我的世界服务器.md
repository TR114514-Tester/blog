下载好后就要启动核心了，按下`Win + R`打开`运行`，输入`cmd`
会出现一个黑色的窗口，输入下面的

```
java -version
```
如果有信息类似于下面这样就行
```
C:\Users\admin>java -version
openjdk version "25.0.1" 2025-10-21 LTS
OpenJDK Runtime Environment Zulu25.30+17-CA (build 25.0.1+8-LTS)
OpenJDK 64-Bit Server VM Zulu25.30+17-CA (build 25.0.1+8-LTS, mixed mode, sharing)
```

接下来把下载下来的核心重命名为`server.jar`
其他的也可以，例如：`a.jar`,`b.jar`，上面纯属是因为好记
创建一个文件夹，但是不要有中文，比如`C:/Users/admin/服务器/`
应该修改为`C:/Users/admin/Server/`
把我们的`server.jar`放到这个文件夹里面
依旧`Win + R`打开运行，输入`cmd`打开CMD
输入下面的，`cd `后面的是你的核心存放的文件夹

```
cd C:/Users/admin/Server
```

接下来我们在里面输入

```
java -jar server.jar
```

后面你会看到一串黄色的东西
